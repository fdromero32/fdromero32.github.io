Tiqqit.net

Î±2020 Capstone Project

Tiqqit.net is a minimalistic open source bug tracking tool, allowing its users to create, edit, delete tickets, and upload images. Built on React on Rails, it features Devise as its account authorization and authentication along with Giphy API for some fun easter eggs.

It was designed and developed by Rudy Becker, Tsz Li, Danny Romero and Andee Isaacs as their LEARN Academy 2020 Alpha capstone project.
