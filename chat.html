Multi User Chat App

AOL Simulator v2020

This project was created using React, Socket.io, and Node.js.

I wanted to learn more about Node.js while reviewing topics in React. In this project, I learned about React Hooks and Functional Components and how to applu them. In the code, I also learned about file structuring in terms of Front End Components as well as separating Front End and Back End files in JavaScript.

Lastly, I incorporated Socket.io, which is a JavaScript library that enables quick data transfer in the backend which allows this app to have multiple users and update the chat in real time.
